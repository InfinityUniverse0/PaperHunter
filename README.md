
| Flag      | Description                                      |
|-----------|--------------------------------------------------|
| `-con`    | One or more conference abbreviations (or `all`)  |
| `-year`   | A specific year or a range (e.g., `2021-2023`)   |
| `-kw`     | At least one keyword must appear (OR)            |
| `-kw_all` | All keywords must appear (AND)                   |

#### Examples


```bash

```
